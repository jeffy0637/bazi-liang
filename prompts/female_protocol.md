# 女命協議

## 概述

女命論斷與男命有顯著差異，本協議規範女命特殊處理方式。

**重要原則**：
- 女命不是「男命的例外」，而是獨立的論命體系
- 避免性別偏見，客觀陳述傳統觀點
- 現代社會變遷，傳統論斷需要調整解讀

## 官殺混雜

### 傳統觀點

```
女命官殺混雜，感情複雜，婚姻多變。
```

### 現代解讀

```
官殺混雜反映：
1. 人生選擇多，需要決斷力
2. 感情經歷豐富（不一定是負面）
3. 職場上有較多挑戰
```

### 制殺與制官

| 情況 | 解讀 |
|------|------|
| 制殺留官 | 剋制情人，留住丈夫 → 感情專一 |
| 制官留殺 | 不重正式婚姻，可能晚婚或特殊婚姻形式 |
| 官殺都不制 | 感情複雜，多情 |
| 官殺都被制 | 感情淡薄，或無緣婚姻 |

## 傷官見官

### 傳統觀點

```
女命傷官見官，剋夫。
```

### 現代解讀

```
傷官見官在女命：
1. 個性獨立，不依附丈夫
2. 可能晚婚或找年齡差距較大的配偶
3. 事業心強
4. 需要尊重和空間的婚姻關係
```

### 有印制傷

```
若有印星制傷官：
- 傷官不傷官
- 化解對丈夫的剋制
- 有長輩或貴人幫助感情
```

## 從兒格（女命）

### 定義

女命食傷旺，日主弱無根，從食傷而去。

### 特點

1. **子女緣深**：一生圍繞子女
2. **才華出眾**：藝術、表達能力強
3. **母愛光輝**：母性特質突出

### 注意

從兒格女命不宜見印星運，印剋食傷，不利子女。
