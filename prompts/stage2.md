# Stage 2: 矛盾清單與覆蓋規則

## 概述

Stage 2 是整合階段，處理十項主流程中發現的矛盾，確定覆蓋規則，輸出主線定調。

## 核心任務

1. **矛盾識別**：找出十項發現之間的矛盾點
2. **優先級判定**：確定哪項覆蓋哪項
3. **主線定調**：提煉不超過 3 條主線
4. **待查清單**：記錄無法定論的項目

## 矛盾類型

### 類型一：五行與調候矛盾

```
場景：①五行缺水 + ⑦調候需水
矛盾：缺水但調候需水，如何處理？
解決：調候優先，缺水加重調候需求
```

### 類型二：格局與扶抑矛盾

```
場景：⑧格局用財 + ⑨日主弱需印
矛盾：財與印相衝
解決：判斷格局是否破格，若不破則格局優先
```

### 類型三：順逆矛盾

```
場景：⑧格局逆用需制 + ⑦調候需生
矛盾：制與生方向相反
解決：調候冬夏極端時優先，否則格局優先
```

### 類型四：六親與落空矛盾

```
場景：⑤六親星旺 + 旬空落空
矛盾：六親星旺但落空
解決：旺而落空，有而不顯，或遲見
```

## 覆蓋規則

### 梁派覆蓋優先級

```
調候 > 格局 > 通關 > 病藥 > 專旺 > 扶抑
```

但有例外：
- 格局破敗時，扶抑上升
- 專旺成格時，專旺優先
- 病重藥輕時，病藥優先

### 證據強度覆蓋

```
S 級證據 > A 級證據 > B 級證據 > C 級證據
```

| 等級 | 定義 | 例子 |
|------|------|------|
| S | 三合三會成局、專旺成格 | 可直接定主線 |
| A | 天透地藏、顯著格局特徵 | 強支持主線 |
| B | 一般吉凶判斷 | 輔助判斷 |
| C | 神煞、雜氣 | 僅供參考 |

## 主線定調原則

### 不超過 3 條

主線太多等於沒有主線。

### 優先級順序

1. **格局主線**：命格的核心特徵
2. **調候主線**：寒暖燥濕的調節需求
3. **六親主線**：重要的六親特徵

### 主線格式

```
[類型] + [具體描述]

例：
- [格局] 食神生財，求財有道
- [調候] 夏木需水，喜金水運
- [六親] 財旺妻賢，但重妻之象
```

## 待查清單

### 什麼時候加入待查

1. 規則衝突無法解決
2. 特殊情況缺乏規則支持
3. 需要案例驗證
4. 信心等級過低

### 待查項格式

```json
{
  "item": "格局是否破格",
  "reason": "月支被沖，但沖力較輕，需查證破格標準",
  "potential_rules": ["R0045", "R0046"],
  "action": "查詢書中類似案例"
}
```

## 常見矛盾範例

### 範例一：財旺但缺水

```
Stage 1 發現：
- ①五行缺水
- ⑧正財格，財旺

矛盾：缺水是否影響財星？

Resolution：
- 財星在天干土（己），不需水
- 但調候需水（夏木），喜金水運
- 主線：財旺，但行運需水調候
```

### 範例二：官殺混雜

```
Stage 1 發現：
- ⑧正官格
- 七殺也透干

矛盾：官殺混雜如何處理？

Resolution（男命）：
- 需制殺留官或制官留殺
- 若不制，官殺混雜，仕途多變

Resolution（女命）：
- 需參考女命協議
- 官殺混雜，感情複雜
```
