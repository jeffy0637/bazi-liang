# 婚姻感情專論

## 婚姻宮與配偶星

### 配偶星（天干）

| 性別 | 配偶星 | 說明 |
|------|--------|------|
| 男命 | 正財 | 正妻、妻子 |
| 男命 | 偏財 | 情人、小三 |
| 女命 | 正官 | 正夫、丈夫 |
| 女命 | 七殺 | 情人、非正緣 |

### 婚姻宮（日支）

日支代表「婚姻宮」，是配偶的位置：
- 日支的五行、十神反映配偶特質
- 日支受刑沖合害影響婚姻穩定

---

## 婚姻吉利組合

### 一、配偶星純一

**男命**：只有正財，無偏財
- 感情專一
- 婚姻穩定
- 不易有外遇

**女命**：只有正官，無七殺
- 丈夫專情
- 婚姻和諧
- 無競爭者

### 二、配偶星得位

配偶星在日支（婚姻宮）：
- 配偶與自己緣分深
- 婚姻感情穩定
- 易得良緣

### 三、日支為用神

日支是命局的用神：
- 配偶對自己有幫助
- 婚後運勢提升
- 夫妻同心

### 四、夫妻宮安穩

日支不受刑沖：
- 婚姻宮穩定
- 感情少波折
- 能白頭偕老

---

## 婚姻不利組合

### 一、配偶星混雜

**男命**：正財、偏財皆有
- 感情不專一
- 易有三角戀
- 婚姻不穩定

**女命**：正官、七殺皆有（官殺混雜）
- 感情複雜
- 婚姻易變
- 配偶可能有外遇

### 二、配偶星被合

配偶星與他支相合：
- 配偶易被他人吸引
- 感情有競爭者
- 需防第三者

### 三、日支逢沖

日支被其他地支相沖：
- 婚姻動盪
- 夫妻分離
- 聚少離多

### 四、婚姻宮空亡

日支落空亡：
- 配偶緣分薄
- 婚姻較晚
- 感情不穩

---

## 男命婚姻分析

### 看正財

1. **正財旺**：妻緣佳，易娶好妻
2. **正財弱**：妻緣薄，晚婚
3. **正財被剋**：婚姻有阻
4. **正財被合**：妻被競爭

### 看偏財

1. **無偏財**：感情專一
2. **偏財多**：風流多情
3. **偏財透干**：易有外遇
4. **偏財在日支**：可能再婚

### 財星與比劫關係

| 組合 | 意義 |
|------|------|
| 比劫奪財 | 兄弟爭妻，感情競爭 |
| 財星被合 | 妻被他人追求 |
| 財星入墓 | 妻體弱或晚婚 |

---

## 女命婚姻分析

### 看正官

1. **正官旺**：夫緣佳，易嫁好夫
2. **正官弱**：夫緣薄，晚婚
3. **正官被剋**：婚姻有阻
4. **正官被合**：夫被競爭

### 看七殺

1. **無七殺**：感情單純
2. **七殺多**：感情複雜
3. **七殺透干**：易有情人
4. **官殺混雜**：婚姻多變

### 食傷與官殺關係

| 組合 | 意義 |
|------|------|
| 傷官見官 | 剋夫，婚姻不順 |
| 食神合殺 | 化解不良姻緣 |
| 傷官制殺 | 以才華服夫 |

## 婚姻時機

### 婚姻運到

1. **流年見配偶星**：遇正緣
2. **流年合日支**：動婚姻宮
3. **大運遇桃花**：感情機會多

### 婚姻宮被沖

日支被流年沖動：
- 未婚者：可能結婚
- 已婚者：婚姻變動

### 配偶星透出

原局配偶星藏干，流年透出：
- 緣分出現
- 可能遇見對象

根據年支或日支查桃花：

| 年支/日支 | 桃花 |
|-----------|------|
| 申子辰 | 酉 |
| 寅午戌 | 卯 |
| 亥卯未 | 子 |
| 巳酉丑 | 午 |

### 桃花類型

| 位置 | 名稱 | 意義 |
|------|------|------|
| 年支 | 牆外桃花 | 早年桃花多 |
| 月支 | 牆內桃花 | 正當姻緣 |
| 日支 | 夫妻桃花 | 配偶有魅力 |
| 時支 | 牆外桃花 | 晚年桃花多 |

## 婚姻建議

### 身強者

- 配偶宜柔順
- 婚後需收斂
- 避免太強勢

### 身弱者

- 配偶宜有擔當
- 需借助伴侶
- 選擇能力強者

### 官殺混雜（女命）

- 感情需專一
- 避免三心二意
- 晚婚更穩定

### 比劫多（男命）

- 需防爭奪
- 感情需用心
- 避免兄弟介入
