# 大運流年推算

## 基本概念

**大運**：每10年一運，主導該時期的總體運勢
**流年**：每1年的運勢，受當年干支影響
**流月**：每月運勢（較少使用）
**流日**：每日運勢（更少使用）

---

## 大運排法

### 起運歲數計算

#### 順逆規則

| 年干陰陽 | 性別 | 排法 |
|----------|------|------|
| 陽年（甲丙戊庚壬） | 男 | 順排 |
| 陽年 | 女 | 逆排 |
| 陰年（乙丁己辛癸） | 男 | 逆排 |
| 陰年 | 女 | 順排 |

#### 計算方法

1. **找節氣**：確定出生日前後的節氣
2. **計算天數**：
   - 順排：從出生日到下一個節氣的天數
   - 逆排：從出生日到上一個節氣的天數
3. **換算歲數**：每3天為1歲
   - 公式：起運歲數 = 天數 ÷ 3（四捨五入）

#### 簡化算法

若無法精確計算，可用以下估算：
- 月初出生（1-10日）：約 3-4 歲起運
- 月中出生（11-20日）：約 5-6 歲起運
- 月末出生（21-31日）：約 7-8 歲起運

---

### 大運干支

從月柱開始，按順逆排列：

**順排**：
```
月柱 → +1 → +2 → +3 → ... （干支各加1）
```

**逆排**：
```
月柱 → -1 → -2 → -3 → ... （干支各減1）
```

#### 範例

若月柱為「丙寅」，順排：
| 運次 | 干支 | 年齡段 |
|------|------|--------|
| 第一運 | 丁卯 | 5-14歲 |
| 第二運 | 戊辰 | 15-24歲 |
| 第三運 | 己巳 | 25-34歲 |
| 第四運 | 庚午 | 35-44歲 |
| 第五運 | 辛未 | 45-54歲 |
| 第六運 | 壬申 | 55-64歲 |
| 第七運 | 癸酉 | 65-74歲 |
| 第八運 | 甲戌 | 75-84歲 |

## 大運分析

### 天干主前五年，地支主後五年

傳統上認為：
- **前5年**：以大運天干為主
- **後5年**：以大運地支為主

### 大運與命盤關係

| 關係 | 吉凶 | 說明 |
|------|------|------|
| 大運為用神 | 大吉 | 運勢順遂，事業有成 |
| 大運為喜神 | 吉 | 運勢平順，有所進展 |
| 大運為閒神 | 平 | 運勢平淡，無大起伏 |
| 大運為忌神 | 凶 | 運勢不順，多有阻礙 |
| 大運為仇神 | 大凶 | 運勢艱難，需謹慎 |

### 大運與原局作用

1. **大運沖原局**：變動大，可能破壞原有格局
2. **大運合原局**：可能增強或減弱某五行
3. **大運刑原局**：易有災禍、是非
4. **大運害原局**：暗中不利、損失

## 流年分析

### 流年干支

流年就是當年的干支，例如：
- 2024年：甲辰年
- 2025年：乙巳年
- 2026年：丙午年

### 流年與命盤關係

| 關係 | 吉凶 | 影響 |
|------|------|------|
| 流年為用神 | 吉 | 當年順利 |
| 流年為忌神 | 凶 | 當年多事 |
| 流年沖命盤 | 變動 | 工作、住所等變化 |
| 流年合命盤 | 緣分 | 感情、合作機會 |

### 流年與大運組合

流年需結合大運來看：

| 組合 | 結果 |
|------|------|
| 大運吉 + 流年吉 | 大吉大利 |
| 大運吉 + 流年凶 | 吉中有凶 |
| 大運凶 + 流年吉 | 凶中有吉 |
| 大運凶 + 流年凶 | 需特別注意 |

## 重要運勢節點

### 換運年

大運交接的年份，運勢可能有明顯變化。

### 太歲年

流年地支與年支相同，俗稱「本命年」：
- 可能有較大變動
- 不一定是凶年
- 需看整體格局

### 犯太歲

| 類型 | 條件 | 影響 |
|------|------|------|
| 值太歲 | 年支=流年支 | 本命年，變動大 |
| 沖太歲 | 年支與流年支相沖 | 衝突、變動 |
| 刑太歲 | 年支與流年支相刑 | 是非、災禍 |
| 害太歲 | 年支與流年支相害 | 暗損、小人 |
| 破太歲 | 年支與流年支相破 | 破壞、損失 |

### 六沖對照

| 沖 | 地支 |
|----|----|
|子午沖|子↔午|
|丑未沖|丑↔未|
|寅申沖|寅↔申|
|卯酉沖|卯↔酉|
|辰戌沖|辰↔戌|
|巳亥沖|巳↔亥|

## 重要流年組合

### 歲運並臨

大運地支與流年地支相同：
- 力量加倍
- 吉則大吉，凶則大凶

### 歲運相沖

大運地支與流年地支相沖：
- 變動劇烈
- 需特別注意

### 歲運天剋地沖

大運與流年天干相剋、地支相沖：
- 極為不利
- 需謹慎行事

## 運勢預測要點

### 看層次

1. **先看大運**：決定10年基調
2. **再看流年**：決定當年吉凶
3. **參考流月**：決定起伏時間

### 看組合

1. 大運與原局的關係
2. 流年與原局的關係
3. 大運與流年的關係

### 看用神

1. 運中有無用神
2. 用神是否受制
3. 忌神是否發動

### 看刑沖合害

1. 有無重大刑沖
2. 合化後的五行
3. 對原局的影響
