# 梁派 Step3：拱 / 夹 / 暗拱（可验证规则）

梁派强调：第三步看拱夹；且暗拱若与原局相冲要警惕（可能激化冲突）。

## 地支顺序（用于夹）
子 丑 寅 卯 辰 巳 午 未 申 酉 戌 亥（循环）

## 夹（夹中间一支）
定义：若同时出现「前一支」与「后一支」，则形成「夹中间那一支」。
输出格式（必须）："{左}{右}夹{中}"
例：丑与卯同时出现 → 丑卯夹寅

## 拱（三合缺一支）
只允许四组三合（缺一支才叫拱）：
- 申子辰（拱水）
- 亥卯未（拱木）
- 寅午戌（拱火）
- 巳酉丑（拱金）

输出格式（必须）："{两支}拱{缺支}"
例：申+辰 → 申辰拱子

## 禁止事项
- 任何不属于「夹」或「三合拱」的，一律禁止写成“拱/夹”。
- 例如：子卯刑不能写成“子卯拱寅”（刑就是刑，不是拱）。
