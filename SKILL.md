# 八字命理占卜技能 (Bazi Mingli Divination Skill)

八字命理（四柱推命）專業排盤與解命系統。源自中國傳統命理學，以出生年月日時的天干地支組合，推算人生命運與運勢。

## 梁派硬规则（本 skill 必须遵守）

1) 必须按梁派顺序：Step1 五行全不全(含支藏) → Step2 刑冲合会 → Step3 拱/夹/暗拱。不得跳步。
2) Step3 的拱/夹必须按 @references/liang-gongjia.md 的"可验证规则"输出：
   - findings.arch_clamp 必须是【字符串数组】
   - 每条必须是标准格式：丑卯夹寅 / 申辰拱子 这种
   - 禁止把"刑"(如子卯刑)写成"拱/夹"
3) 若暗拱/夹得出的暗支与原局形成相冲（或重大刑），必须在 notes 标记"梁派警示"。

當用戶請求以下類型的幫助時，應啟用此技能：

1. **排盤相關**：請求八字排盤、命盤查詢、生辰八字分析

3. **運勢預測**：事業財運、感情婚姻、健康運勢
4. **關鍵詞觸發**：「八字」「四柱」「命盤」「排盤」「算命」「命理」「五行」「大運」「流年」

## 解盤流程

依梁系流程執行，詳見 @references/liang-system.md：

### Step 0：排盤校驗（工具層）
- 執行 `scripts/bazi_engine.py` 或 `scripts/bazi_calc.py`
- 輸出：四柱、藏干、十神、五行分布

### Step 1：五行是否全（含支藏）
- 判斷五行缺失
- 標註缺失的六親/層次影響

### Step 2：刑沖合會
- 描述結構，不先急著下吉凶
- 標記合會成局/成格

### Step 3：拱、夾、暗拱
- 找出拱夾暗拱
- 檢查危險組合

### Step 4-7：解讀層（LLM 負責）
- Step 4：十神定位（性格/行為傾向）
- Step 5：格局與職業傾向
- Step 6：調候（寒暖燥濕）
- Step 7：流年/大運架構

## 倫理準則

詳見 @ETHICS.md，核心要點：

- **中立客觀**：吉凶並陳，不迎合期望
- **責任界限**：僅供參考，不替代專業諮詢
- **語言規範**：用「傾向」「可能」，避免「一定」「絕對」
- **他人隱私**：未經同意不分析他人完整命盤
- **情緒脆弱者**：強調僅供參考，鼓勵專業支持
- **自傷相關**：不排盤，表達關心，提供危機資源

## 常見問題處理

| 情況 | 處理方式 |
|------|----------|
| 時辰不確定 | 詢問大致範圍，說明可能差異 |
| 只知農曆 | 需轉換為西曆以確定節氣 |
| 問他人八字 | 提醒需當事人同意，僅供參考 |
| 對結果不滿 | 強調命理是趨勢非定數，重在自我提升 |
| 反覆詢問同一問題 | 提醒命盤不變，建議接受分析結果 |
