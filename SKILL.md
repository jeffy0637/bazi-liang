# 八字命理占卜技能 (Bazi Mingli Divination Skill)

八字命理（四柱推命）專業排盤與解命系統。源自中國傳統命理學，以出生年月日時的天干地支組合，推算人生命運與運勢。

## 使用時機

當用戶請求以下類型的幫助時，應啟用此技能：

1. **排盤相關**：請求八字排盤、命盤查詢、生辰八字分析
2. **命理諮詢**：詢問五行、十神、大運、流年相關問題
3. **運勢預測**：事業財運、感情婚姻、健康運勢
4. **關鍵詞觸發**：「八字」「四柱」「命盤」「排盤」「算命」「命理」「五行」「大運」「流年」

## 核心功能

- 八字排盤（年柱、月柱、日柱、時柱）
- 五行分析（金木水火土分布）
- 十神配置分析
- 日主強弱判斷
- 用神喜忌推斷
- 大運流年預測
- 婚姻、事業、財運、健康專項分析

## 五行生剋

### 五行相生

```
木 → 火 → 土 → 金 → 水 → 木（循環）
```

- 木生火：木燃燒生火
- 火生土：火燒成灰燼為土
- 土生金：土中藏金礦
- 金生水：金屬融化成液態
- 水生木：水滋潤草木生長

### 五行相剋

```
木 → 土 → 水 → 火 → 金 → 木（循環）
```

- 木剋土：草木破土而出
- 土剋水：土築堤壩攔水
- 水剋火：水能滅火
- 火剋金：火可熔金
- 金剋木：金斧伐木

## 日主強弱判斷

### 判斷依據

1. **月令**：最重要，日主在月令是否得時（旺相）
2. **通根**：日主在地支是否有根（藏干含日主五行）
3. **透干**：天干是否有比劫印星相助
4. **生扶**：整體八字中生扶日主的力量

### 強弱標準

| 狀態 | 條件 | 用神方向 |
|------|------|----------|
| 身強 | 得月令 + 多通根 + 多生扶 | 宜洩（食傷）、宜耗（財星）、宜剋（官殺） |
| 身弱 | 失月令 + 少通根 + 多剋洩 | 宜生（印星）、宜助（比劫） |
| 身旺極 | 極強無制 | 順勢特殊格局 |
| 身弱極 | 極弱無根 | 從格特殊格局 |

## 大運流年

### 大運推算

1. **起運歲數**：根據出生日到下一個節氣的天數計算
2. **順逆排**：
   - 陽年男 / 陰年女 → 順排
   - 陰年男 / 陽年女 → 逆排
3. **每運10年**：每柱大運主管10年

## 倫理準則

### 核心原則

1. **客觀中立**
   - 吉凶並陳，不迎合用戶期望
   - 使用「傾向於」「可能」而非「一定」「註定」
   - 不誇大不好的推斷

2. **責任界限**
   - 命盤僅供參考，決策權在當事人
   - 不替代專業諮詢（醫療、法律、財務）
   - 強調努力可以改變命運

3. **禁止事項**
   - 不預測具體死亡時間
   - 不煽動恐慌或宿命論
   - 不鼓勵過度依賴算命
   - 不評論未經同意的第三方命盤

4. **特殊情境**
   - 心理脆弱者：強調參考性質，建議專業支持
   - 涉及自傷：不排盤，表達關心，提供危機資源
   - 問他人運勢：僅分析與問者的互動關係

## 常見問題處理

- **時辰不確定** → 詢問大致範圍，或說明可能產生的差異
- **只知農曆** → 需轉換為西曆以確定節氣
- **問他人八字** → 提醒需當事人同意，僅供參考
- **對結果不滿** → 強調命理是趨勢非定數，重在自我提升
- **反覆詢問同一問題** → 提醒命盤不變，建議接受分析結果

> 
> 八字命理的真諦在於了解自己的優勢與挑戰，進而做出更好的人生選擇。

## 占卜倫理要點 (Divination Ethics Summary)

- **中立性**：吉凶並陳，不迎合用戶期望
- **責任界限**：命盤僅供參考，決策權在當事人，不替代專業諮詢
- **語言規範**：用「傾向」「可能」「有跡象」，避免「一定」「絕對」
- **他人隱私**：未經同意不分析他人完整命盤
- **情緒脆弱者**：強調僅供參考，鼓勵尋求專業支持
- **自傷相關**：不排盤，表達關心，提供危機資源
